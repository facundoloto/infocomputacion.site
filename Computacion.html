NAVEGAR POR EL ESCRITORIO

El escritorio de Windows es aquella interface de software que ha sido originalmente creada con el objetivo de generar un espacio de cómodo y fácil acceso a los programas y operaciones disponibles de una computadora. Definicion de escritorio en windows

  1. El escritorio es la parte principal de un sistema operativo a lo que se refiere visualmente ya que en el es donde esta las funciones basicas con la que interactua el usuario.

  2. la barra de tareas sirve para controlar los programas que estan en usos y las tareas activas.

  3. Un icono es una pequeña imagen que está asociada a un elemento de Windows,un programa,una herramienta o a un archivo, y que haciendo doble clic en él nos permite abrir dichos elementos, programas o archivos o en el caso de una herramienta aplicar su función,si esta tiene una flecha significa que es un acceso directo a ese programa.

  4. Es una lista gráfica de accesos directos a diversas funciones como los programas u opciones comunes como Documentos y Apagar el sistema.

  5. La papelera de reciclaje es donde va a parar todo los archivos que eliminamos.

  6. Todo programa que abramos aparecera el icono con un recuadro transparente en la barra de tareas significa que esta en uso el programa.

  7. La tareas activas estan en la parte inferior del lado derecho de la barra de tareas en el podemos encontrar la conexion de wifi,la hora y el antivirus por ejemplo,estas funciones estan siempre activas.

    8. MENU DE INICIO

    En el menu de inicio hay varias funciones importantes como la opcion de apagado,suspencion y reinicio,sino que se encuentra el panel de control o opciones,y es donde se configura y personaliza todas las opciones del sistema operativo desde personalizar el fondo del escritorio hasta poder configurar las opciones de seguridad,todo esto lo vamos a ver acontinuacion.

    1. tenemos un buscador del sistema operativo donde si queremos buscar un programa o un archivo que tengamos y no encontramos lo buscamos desde ahi.

    2. estas son conocidas como opciones de energia en las cuales estan la opciones de apagado,suspender/hibernar,reinicio,bloqueo,cambio de usuario,cerrar sesion.

      1. apagado:obviamente esta opcion tal cual su nombre indica apaga el sistema

      2. hibernar/suspender:la opcion esta hace que el sistema se apague pero sin cerrar ningun programa que estes usando y al momento de prenderlo de vuelta reanuda la actividad.

      3. reinicio:esto hace que el sistema cierre todos los programas activos y la sesion y vuelve a reinicar el sistema como cuando se prende.

      4. bloqueo:bloquea la sesion del sistema pero toda su actividad no se suspenden ni se apagan,sigue funcionando normalmente,solamente bloquea el inicio de secion y hasta entremos de vuelta.

      5. cambio de usuario:esto sirve si un ordenador hay varias cuentas para entrar al sistema operativo si alguien quiere entrar a otro usuario solo basta con esta opcion sin necesidad de reiniciar o apagar el sistema.

      6. cerrar sesion:esto hace que se cierren toda las actividades y vuelve al inicio de sesion.
